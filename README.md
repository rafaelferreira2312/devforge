# 🔨 DevForge – Central de setups e produtividade para engenheiros

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](CONTRIBUTING.md)

> **Forge your development environment** – Scripts, documentação, troubleshooting e recomendações de hardware para mais de 15 stacks.

## ✨ Características
- ⚡ Instalação com 1 comando (Linux/Win/Mac)
- 🎨 UI moderna inspirada em Vercel, Linear, Warp
- 🌙 Dark mode nativo
- 🔍 Busca global por stacks e comandos
- 📋 Botão copiar em todos os comandos
- 🐳 Docker opcional em cada stack

## 🚀 Como usar
1. Clone o repositório: `git clone https://github.com/devforge/devforge`
2. Abra `index.html` no navegador (ou use Live Server)
3. Explore stacks via sidebar ou cards

## 📦 Estrutura
- `stacks/` – Páginas individuais de cada tecnologia
- `scripts/` – Shell/PowerShell para automação
- `assets/` – CSS, JS modular
